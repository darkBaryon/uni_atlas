# 中国留学生主要就读的国外大学名单

> 整理日期：2026-07-03。数据来源：美国《门户开放报告》(Open Doors 2025)、英国 HESA 2023/24 学年数据、各留学机构统计。
> 总体格局：美国约 26.6 万中国留学生（生源国第二，仅次于印度）、英国 15 万+、澳大利亚约 16 万、加拿大约 8.5 万、新加坡约 3.8 万。港新日等亚洲目的地近年热度持续上升。

## 趋势要点（供咨询参考）

1. 2025 年中国出国留学总人数约 57 万，较峰值回落；赴美人数从 2019 年约 37 万峰值降至 26.6 万。
2. 英国已超越美国成为中国学生首选目的地（British Council）。
3. 港、新、马、日及欧洲因成本低、距离近、政策稳定，热度快速上升。
4. 澳、加均在收紧国际学生名额，申请竞争与政策风险需提示学生。
5. 专业趋势：STEM（数学/CS、工程）热度超过商科，但商科在英澳港仍是最大品类（约 39%）。
